# Frontend

Das Frontend ist der Teil einer Anwendung, mit dem Benutzer direkt interagieren. Es umfasst die Benutzeroberfläche, die Darstellung von Inhalten und die Interaktion mit dem Backend. Moderne Frontend-Entwicklung nutzt häufig Frameworks wie React, Angular oder Vue.js, um dynamische und reaktive Benutzeroberflächen zu erstellen.

Frontend-Entwicklung geschieht in dem lange bekannten Dreier: HTML, CSS und TypeScript. Auf diesem Teil der Webentwicklung liegt bislang mein größter Fokus. Daher habe ich damit die meiste Erfahrung.

In dieser Sektion möchte ich die Begriffe, Technologien und Probleme (in Form fieser Bugs) näher beschreiben, die mir bei der Arbeit in dem Bereich begegnen.
