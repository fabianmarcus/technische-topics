# Künstliche Intelligenz

Der Begriff Künstliche Intelligenz wird mittlerweile sehr undifferenziert verwendet. Was KI genau ist oder nicht ist, möchte ich hier nicht erläutern. Ich möchte hier nur auf KI im Rahmen meiner Arbeit als praktischer Webentwickler eingehen. Dazu werde ich die Komponenten erläutern, die in der Praxis eine Rolle spielen und in ihrer Gesamtheit "Künstliche Intelligenz" ausmachen, wie sie zurzeit im Internet gehandelt wird. Ich werde auch auf die Grenzen von KI eingehen und erläutern, was einem bei der Nutzung klar sein muss.

## Grundlagen-Buzzwords

| Begriff | Erklärung |
| --- | --- |
| [LLM](ki.llm.md) | Large Language Model. Das Gehirn der KI, das die Daten enthält. |
| [Token](ki.llm.md) | Die Informationseinheit eines LLM. |
| [Tokenisierung](ki.llm.md) | Prozess, bei dem ein Text in Token zerlegt wird, um ihn für das LLM verarbeitbar zu machen. |
| [Vektor](ki.llm.md) | Mathematische Darstellung von Informationen (Beispiel: "Hund" = [0.1, 0.2, 0.3, 0.4]) |
| [Embedding](ki.llm.md) | Vektorisierung der Informationen, um sie mathematisch vergleichen zu können. |
| [Gewichtung](ki.llm.md) | Maß für die Relevanz von Informationen, die in den Trainingsdaten enthalten sind. |
| [NLP](ki.llm.md) | Natural Language Processing, die Transformation von natürlicher Sprache in mathematische Vektoren und umgekehrt. |
| [Trainingsdaten](ki.llm.md) | Strukturierte Informationen, die zum Trainieren von KI-Modellen verwendet werden. |
| [RAG](ki.rag.md) | Retrieval Augmented Generation, eine Methode, bei der KI-Modelle auf externe Daten zugreifen, um ihre Antworten zu verbessern. |
| [Prompt](ki.llm.md) | Eingabeaufforderung, die an ein KI-Modell gesendet wird, um eine Antwort zu generieren, unterteilbar in System- und User-Prompt. |
| [Halluzination](ki.halluzination.md) | Wenn ein KI-Modell eine Antwort generiert, die nicht auf den Trainingsdaten basiert und somit faktisch falsch ist. |
| [MCP](ki.mcp.md) | Model-Context-Protocol, ein Protokoll, das die Kommunikation zwischen dem KI-Modell und Agenten steuert. |
| [F&A](ki.qa.md) | Fragen & Antworten - ein Format, bei dem ich einige Fragen an Cope hier festhalte. |
